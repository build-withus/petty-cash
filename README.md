#Petty Cash Management System

A web-based Petty Cash Management System built with Laravel to simplify the management of petty cash within an organization. The application enables secure tracking of income, expenses, balances, and financial records through an intuitive dashboard.

## Features

- User authentication
- Dashboard overview
- Record income and expenses
- Track petty cash balance
- Transaction history
- Expense categorization
- Financial reports
- Role-based access control
- Responsive user interface

## Tech Stack

- Laravel
- PHP
- MySQL
- Tailwind CSS
- Vite

## Installation

```bash
git clone https://github.com/yourusername/petty-cash.git

cd petty-cash

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm install

npm run dev

php artisan serve
```
